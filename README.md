# ModVerse

An open-source, moddable sandbox game with a custom-built HTML5/WebGL engine — no Unity, Unreal, Godot, or other game engine involved. Ships as a Progressive Web App and as an Android APK via Capacitor.

## Highlights

- **Custom engine**: fixed-timestep loop, WebGL2 batched sprite renderer, top-down tile physics, chunk-streamed infinite world, all hand-written (see `public/js/engine`, `public/js/renderer`, `public/js/physics`).
- **Infinite world**: seeded noise-based terrain generated on Web Workers, streamed in/out around the player (`public/js/world`).
- **Gameplay**: movement, inventory + crafting, NPCs with simple AI, mountable vehicles, day/night cycle, dynamic weather.
- **Mod system**: drop-in mods with a `manifest.json` + JS API, executed in sandboxed Web Workers with per-mod permissions. Auto-loaded from `/storage/emulated/0/ModVerse/` on Android.
- **Dev console**: `spawn`, `give`, `tp`, `weather`, `time`, `reloadmods`, extensible by mods.
- **Multiplayer**: WebSocket relay for player state/chat sync.
- **PWA**: offline caching, background sync for autosave, push notifications.
- **Backend**: Express + SQLite for accounts, save slots, and persisted world edits.

## Repository layout

```
ModVerse/
├── server/            Express + SQLite backend (auth, saves, chunk persistence, mod listing, WS relay)
├── public/             The PWA client (also what Capacitor packages into the Android app)
│   ├── js/engine        Loop + Engine orchestrator
│   ├── js/renderer       WebGL2 renderer + procedural texture atlas
│   ├── js/world          Chunk, ChunkManager, WorldGen (noise), World
│   ├── js/physics        AABB + top-down collision physics
│   ├── js/entities       Entity, Player, NPC, Vehicle
│   ├── js/ui             Inventory, Crafting, HUD, DevConsole, UIManager
│   ├── js/audio          Procedural WebAudio SFX/music manager
│   ├── js/network        REST + WebSocket client
│   ├── js/mods           ModLoader, ModAPI, sandboxed worker runtime
│   ├── js/save           IndexedDB + server-synced save system
│   ├── js/weather        DayNightCycle, WeatherSystem
│   └── js/workers        chunkWorker.js (off-thread terrain generation)
├── mods/               Example mods (used by the dev server AND as templates for real mods)
├── android/            Generated by `npx cap add android` (not checked in until you run it)
├── docs/               Full documentation set (see below)
└── capacitor.config.json
```
